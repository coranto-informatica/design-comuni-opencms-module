**Design Comuni OpenCms** is a module for [OpenCms](https://www.opencms.org/en/) implementing the **official design system for Italian municipalities websites**, in compliance with the [Design Guidelines by Designers Italia](https://designers.italia.it/).

This module provides:

- Templates and graphic resources aligned with official guidelines;
- Ready-to-use configurations for OpenCms environments;
- No editorial content, allowing full customization and reuse by different public bodies.

It enables public administrations to **quickly create accessible, consistent, and compliant municipal websites**, reducing development time and ensuring design uniformity.
